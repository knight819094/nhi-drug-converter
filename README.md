# 健保用藥品項轉換工具

## 專案目的

台灣衛生福利部中央健康保險署每月發布「健保用藥品項」TXT 檔案（固定寬度格式、Big5HKSCS 編碼）。
本工具供藥局人員使用，將每月兩個原始 TXT 檔自動合併、解析、標記，並輸出為格式化 Excel 檔案。

---

## 使用者輪廓

- **使用者**：藥局人員（非工程師背景）
- **使用頻率**：每月一次
- **使用環境**：Mac / Windows 桌面應用程式
- **技術能力**：只會點擊操作，不會使用命令列

---

## 核心流程

```
每月放入 2 個 TXT 檔
        ↓
  開啟工具（GUI）
        ↓
  自動解析、合併、去重
        ↓
  快速篩選 + 劑型篩選
        ↓
  輸出格式化 Excel
```

---

## 功能清單

### 1. 資料解析
- 自動讀取指定資料夾內所有 TXT 檔（每月固定 2 個）
- 編碼：Big5HKSCS（含全形符號自動轉半形，`〝〞` → `""`）
- 格式：固定寬度欄位，自動萃取各欄位
- 兩檔合併後以藥品代碼去重（保留最新生效日；歷史模式則保留全部）
- 中文藥名前置廠商名自動移至末尾
  （`"鴻汶"安保喜樂錠` → `安保喜樂錠 "鴻汶"`）
- 含量／容量去除多餘尾隨零（`5.000 MG/ML` → `5 MG/ML`）
- 廠商名去除「台灣」前綴（`台灣拜耳股份有限公司` → `拜耳股份有限公司`）

### 2. 輸出欄位（依下列順序）
| 欄位 | 說明 |
|------|------|
| 藥品代碼 | 健保署代碼 |
| 中文藥名 | 藥品中文名稱 |
| 建議 | 自動標記（見下方） |
| 健保價格 | 當月有效健保給付價格 |
| 生效日 | 本次健保價格生效日期 |
| 停用日 | 品項停用日期（無停用為 9999/12/31） |
| 英文藥名 | 藥品英文名稱 |
| 主成分 | 有效成分英文名稱 |
| 劑型 | 錠劑、注射劑、膠囊劑⋯等（共 170+ 種） |
| 廠商 | 製造廠商全名 |
| 含量 | 劑量單位（MG、MG/ML、USP-U⋯等） |
| 容量 | 包裝容量（ML、G⋯等） |
| 英文藥名首位 | 英文藥名第一個字（PROPER 格式） |
| 主成分首位 | 主成分第一個字（PROPER 格式） |

### 3. 建議標記
| 標記 | 條件 | 儲存格底色 |
|------|------|-----------|
| 🆕 本月新增 | 生效日在近 N 天內（config 可設定，預設 62） | 淺綠 |
| ❌ 無健保給付 | 健保價格為 0 | 淺橘 |
| ✅ 正常 | 一般有效品項 | 無 |

> 本月新增優先於無健保給付（新增但價格為 0 仍顯示 🆕）

### 4. 快速篩選
分兩行排列：

**第一行（三選一，互斥）**
1. 📋 輸出全部歷史資料 — 保留所有歷史紀錄，不去重
2. 🆕 只輸出當月更新 — 僅保留 🆕 本月新增品項
3. 🔮 匯出半年內生效資料 — 生效日介於今日～今日+183 天（輸出仍為歷史總表）

**第二行（獨立複選）**
1. 📅 只顯示停用日大於今日 — **預設勾選**，過濾已停用品項
2. ❌ 排除無健保給付 — 過濾健保價格為 0 的品項

### 5. 劑型篩選
- 解析完成後列出所有劑型（含各劑型品項數量）
- 三欄排列，可捲動瀏覽
- 支援文字搜尋即時篩選（如輸入「注射」只顯示注射相關劑型）
- 全選 / 全不選快速操作
- 僅輸出勾選的劑型品項

### 6. Excel 輸出格式
- 標題列：深藍底色（#1F4E79）、白色粗體
- 奇偶列交替底色（淺藍 #DDEEFF）
- 建議欄依標記類型著色
- 健保價格：千分位格式，靠右對齊
- 日期格式：YYYY/MM/DD
- 欄寬自動調整（最寬 50，建議欄固定 15）
- 凍結第一列標題
- 啟用 AutoFilter（可直接在 Excel 篩選）
- 包含「摘要」工作表（總品項數、各類別數量、轉換日期）

**輸出檔名規則**（以 2026-06 為例）：
| 篩選狀態 | 檔名 |
|---|---|
| 一般 | `2026-06.xlsx` |
| 輸出全部歷史資料 | `2026-06 歷史資料.xlsx` |
| 只輸出當月更新 | `2026-06 當月更新.xlsx` |
| 匯出半年內生效資料 | `2026-06 半年內生效.xlsx` |
| 額外勾選「排除無健保給付」 | 檔名後加 ` (全給付)` |

### 7. 快速連結
- 🔍 **健保用藥查詢**：點擊開啟健保署線上查詢頁
- ⬇️ **下載更新檔**：點擊開啟健保署 TXT 下載頁

### 8. 歷史紀錄頁籤
- 顯示過往匯出的 Excel 檔清單，可一鍵開啟

### 9. 設定（⚙️ 齒輪 / config.yaml）
可用文字編輯器調整以下設定，不需改程式：
- `ui.font_size`：介面基礎字體大小（所有元件等比例縮放）
- 來源資料夾 / 輸出資料夾路徑
- 本月新增判斷天數（`new_item_days`，預設 62 天）
- Excel 交替列顏色、欄寬上限
- 自動套用系統淺色 / 深色主題

### 互動流程
1. 工具啟動 → 自動掃描 TXT 資料夾 → 劑型清單載入完成
2. （選用）勾選快速篩選條件
3. 使用者在搜尋框輸入關鍵字 → 劑型清單即時篩選
4. 勾選 / 取消需要的劑型
5. 按「開始轉換」→ 進度條啟動（背景執行緒，不卡 UI）
6. 完成 → 彈出完成提示視窗，可直接開啟 Excel

## 技術規格

- **語言**：Python 3.10+
- **GUI 框架**：tkinter（內建，無需額外安裝）
- **主要套件**：pandas、openpyxl、pyyaml
- **來源編碼**：Big5HKSCS
- **執行方式**：`python main.py`
- **跨平台**：自訂彩色按鈕（macOS 相容）、系統主題偵測、跨平台檔案開啟
- **檔案結構**：
  ```
  健保用藥品項轉換/
  ├── main.py        ← 唯一程式檔（GUI + 核心邏輯）
  ├── config.yaml    ← 設定檔
  ├── requirements.txt
  ├── README.md
  ├── 健保藥檔載入/  ← 每月放入 TXT
  ├── 匯出結果/      ← 輸出 Excel
  ├── 封包/          ← Windows 打包相關檔案（見封包/README.md）
  └── .github/workflows/build-windows.yml  ← GitHub Actions 自動打包
  ```

## 打包成 Windows 安裝檔

詳見 [`封包/README.md`](封包/README.md)。提供兩種方式：

1. **GitHub Actions**（推薦）：推上 GitHub → Actions 一鍵自動產出安裝檔
2. **本機打包**：在 Windows 執行 `封包\build_windows.bat` + Inno Setup
